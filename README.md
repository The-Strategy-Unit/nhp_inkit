An R package to handle and visualise data from the inputs data pipeline of the [New Hospital Programme (NHP) demand model](https://connect.strategyunitwm.nhs.uk/nhp/project_information/).
Designed primarily to support the [NHP Inputs](https://github.com/The-Strategy-Unit/nhp_inputs/) and [TPMA explorer](https://github.com/The-Strategy-Unit/tpma-explorer) web applications.
